# NoSQL Analysis Report

## 1. What is NoSQL and MongoDB
NoSQL databases store data in flexible formats like JSON documents.
MongoDB is a document-oriented NoSQL database that allows schema-less
data storage and easy handling of nested data.


## 2. Why MongoDB for Product Catalogs
MongoDB is suitable for product catalogs because it supports nested
structures such as specifications, reviews, and tags. This makes data
storage simple and avoids complex joins, improving performance and
flexibility.


## 3. Aggregation and Updates in MongoDB
MongoDB provides aggregation pipelines to analyze data such as calculating
average ratings and prices. Update operators like $push allow easy
modification of array fields, such as adding new product reviews.
